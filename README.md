# DramaPipeline

> 影视流水线 - 工业级短剧生成工具
>
> 面向专业级用户和企业用户，采用流水线式、每步人工确认的控制台风格 Web 平台。

## 项目定位

- **非全自动**：每个流水线步骤需用户确认输出内容后，再手动分发到下一阶段
- **流水线式**：角色按顺序执行，用户主动触发下一步
- **控制台风格**：Web 平台采用控制台式 UI，偏专业、高效、可观测

## 技术栈（规划中）

- Web：React + Tailwind（控制台基底）
- 流水线：10 个 Agent 角色（剧本分析、AI 导演、分镜设计、视频生成等）
