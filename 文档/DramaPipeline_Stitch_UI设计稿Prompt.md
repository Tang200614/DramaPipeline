## 一、需求摘要

**产品定位**：工业级短剧生成工具，面向短剧创作者、导演、制片人及 MCN、影视公司。非全自动流水线，每步需用户确认后手动分发到下一阶段。

**核心能力**：
- 多剧本并行：支持多剧本同时运行，用户可切换不同剧本
- 剧本输入：用户上传剧本（txt/docx/md）或输入大纲，由 AI 生成剧本
- 素材输入：用户上传 zip、rar、图片
- 自定义角色：用户定义角色，AI 根据角色修改剧本
- 剧本切片：剧本切片后交给分镜设计
- 用户配置：剧本量、拼接字数、匹配章节
- 知识库：用户上传的剧本作为当前项目知识库

**10 个流水线角色**（按顺序执行）：剧本分析机器人、AI 导演团队、素材分析助手、分镜设计助手、首帧生成助手、视频生成助手、后期合成助手、角色创建大师、场景生成大师、审片助手。

**交互原则**：每步输出后暂停 → 用户查看、编辑、确认 → 点击「分发」进入下一角色；支持回退、重做；多项目/多任务并行。

## 二、Stitch 设计稿 Prompt（可直接复制）

```
请为 DramaPipeline 影视流水线 Web 控制台设计 UI 设计稿。

【项目定位】
工业级短剧生成工具，面向专业级用户（短剧创作者、导演、制片人）和企业用户（MCN、影视公司）。非全自动流水线，每步需用户确认后手动分发到下一阶段。

【整体风格】
- 控制台风格：专业、高效、可观测，偏工业感
- 布局：左侧固定导航栏（约 220–240px）+ 右侧主内容区
- 配色：深色主题，主色 electric blue #0080ff，渐变 #0080ff → #00d4ff
- 视觉：玻璃拟态卡片（glass-card）、清晰信息层级、状态标签醒目
- 参考：zeroclaw 控制台布局与深色主题

【导航结构】（左侧 Sidebar）
1. 工作台
2. AI 对话
3. 流水线（核心页面，见下方三块式布局）
4. 任务列表
5. 系统设置

【流水线页面 - 三块式布局】（/pipelines 主内容区）
流水线页面采用三块式布局，嵌入在左侧导航 + Header 的主框架内：

1. 顶部横条（高度 48–56px，全宽）
   - 剧本选择：下拉/标签切换当前剧本
   - 操作：新建剧本、上传剧本、输入大纲生成剧本
   - 当前选中剧本高亮，可显示「运行中」「待确认」等 badge

2. 左侧角色列表（宽度 240–280px，可折叠）
   - 标题/说明：每个剧本有独立的角色组、素材库、数据
   - 10 个角色列表，每项包含：头像/图标 + 名称 + 副标题
   - 角色顺序：剧本分析机器人、AI 导演团队、素材分析助手、分镜设计助手、首帧生成助手、视频生成助手、后期合成助手、角色创建大师、场景生成大师、审片助手
   - 状态区分：待执行、执行中、待确认、已完成（颜色或图标）
   - 点击角色 → 右侧展示该角色的会话与进度

3. 右侧主内容区（flex 自适应）
   - 选中角色的会话：Chat 消息列表，可查看上一角色输出、当前角色回复
   - 处理进度：进度条、状态文案
   - 操作栏：确认按钮、分发到下一阶段、回退、重做
   - 输出预览：文本块、图片缩略图或视频播放器（按角色类型）

【其他页面】
- 工作台：多剧本总览、进行中任务数、待确认步骤数、快捷新建、最近任务列表
- 任务列表：表格/卡片，任务名称、项目、当前步骤、状态、创建时间、操作
- 系统设置：剧本量、拼接字数、匹配章节等配置项

【状态标签】
5 种状态：待执行、执行中、待确认、完成、失败
- 待执行：灰色
- 执行中：蓝色
- 待确认：黄色/琥珀
- 完成：绿色
- 失败：红色

【配色建议】
- 背景主色：#050510 / #0a0a18
- 卡片：rgba 半透明 + 边框（glass 效果）
- 主色：#0080ff（electric blue）
- 成功：#22c55e
- 警告：#f59e0b
- 失败：#ef4444
- 文字主色：#e8edf5
- 文字次要：#556080

【响应式】
- 主要适配桌面端 1280px 及以上
- 左侧角色列表可收折为图标栏（平板）
```
