## 一、导航与页面流转

### 1.1 主导航结构

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 工作台 | 多剧本总览、运行中任务、快捷新建 |
| `/chat` | AI 对话 | 通用 AI 对话 |
| `/pipelines` | 流水线 | 三块式：剧本选择 + 角色流转 + 角色会话 |
| `/tasks` | 任务列表 | 跨剧本任务、状态筛选 |
| `/config` | 系统设置 | 剧本量、拼接字数、匹配章节等配置 |

### 1.2 页面切换

- **过渡动画**：`page-fade`，路由切换时淡入 + 轻微位移（约 0.2s）
- **Layout**：`router-view` 使用 `<Transition name="page-fade">`

### 1.3 跨页面跳转

| 来源 | 目标 | 触发 | 参数 |
|------|------|------|------|
| 任务列表 | 流水线 | 点击「进入流水线」 | `scriptId`、`episodeId` 或 `taskId` |
| 工作台 | 流水线 | 点击最近任务 | 同上 |
| 工作台 | 任务列表 | 点击「查看任务」 | - |
| 流水线 | 任务列表 | Sidebar 切换 | - |

## 二、流水线页面交互

### 2.1 三块式布局

- **顶部**：剧本选择器，切换剧本/集数时左侧角色列表、右侧内容联动更新
- **左侧**：10 角色列表，点击角色 → 右侧展示该角色会话与进度
- **右侧**：选中角色的会话区，未选时显示空态「请从左侧选择角色」

### 2.2 预览模式

- **触发**：点击「预览」按钮
- **效果**：左侧分析助手面板缩小（flex 过渡），右侧产物面板从右滑入（`preview-drawer`）
- **关闭**：点击「关闭预览」或「确认并分发」

### 2.3 操作按钮反馈

| 操作 | 反馈 |
|------|------|
| 确认并分发 | 按钮 loading 态，成功后关闭预览、更新角色状态 |
| 回退 | 二次确认（可选），成功后刷新当前角色输出 |
| 重做 | 同上 |
| 生成视频 | 按钮 loading，完成后展示视频预览 |

### 2.4 流式输出

- **状态**：`loading` 时禁用发送、显示骨架或占位
- **结束**：后端返回 `{"done": true}` 时，前端 `finally` 将 loading 置为 false

## 三、任务列表页面交互

### 3.1 布局（参考流水线）

- **顶部**：筛选栏（剧本、状态、时间、搜索）+ 新建任务
- **左侧**：任务列表，点击任务 → 右侧展示详情
- **右侧**：任务概览 + 操作（进入流水线、暂停、取消）

### 3.2 筛选

- 剧本：下拉多选
- 状态：全部 / 进行中 / 已完成 / 待确认 / 失败
- 时间：最近 7 天 / 30 天（可选）
- 搜索：实时过滤，防抖 300ms

## 四、状态与视觉反馈

### 4.1 流水线角色状态

| 状态 | 颜色 | 图标 | 说明 |
|------|------|------|------|
| 待执行 | 灰 `#9ca3af` | ⏳ | 尚未开始 |
| 执行中 | 蓝 `#0080ff` | ⟳ 旋转 | 正在处理 |
| 待确认 | 琥珀 `#f59e0b` | - | 等待用户确认 |
| 已完成 | 绿 `#22c55e` | ✓ | 已确认通过 |
| 失败 | 红 `#ef4444` | - | 执行失败 |

### 4.2 加载态

- **全局**：顶部进度条或全屏遮罩（按场景）
- **局部**：按钮内 spinner、骨架屏、占位块
- **流式**：逐字显示，光标闪烁

### 4.3 错误态

- **接口失败**：Toast 提示 + 重试按钮
- **表单校验**：行内错误文案，红色边框
- **空态**：插图 + 引导文案 + 操作入口

## 五、动效规范

### 5.1 过渡时长

| 类型 | 时长 | 缓动 |
|------|------|------|
| 页面切换 | 0.2s | ease |
| 面板滑入 | 0.35s | cubic-bezier(0.32, 0.72, 0, 1) |
| 面板滑出 | 0.28s | 同上 |
| 悬停/焦点 | 0.15s | ease |

### 5.2 已实现动效

- `page-fade`：路由切换
- `preview-drawer`：右侧产物面板从右滑入
- `preview-left-panel`：`has-preview` 时 flex 收缩过渡

## 六、响应式

- **桌面**（≥1280px）：三块式完整展示
- **平板**：左侧角色/任务列表可收折为图标栏
- **移动**：考虑底部 Tab 切换「剧本」「角色」「会话」

---

## 七、无障碍（可选）

- 焦点顺序：Tab 导航符合逻辑
- 键盘：支持 Enter 确认、Esc 关闭弹窗
- 色盲：状态除颜色外辅以图标/文字区分
